# TOML Versions

## TOML v1.1.0
[TOML v1.1.0](https://github.com/toml-lang/toml/issues/928) has not been officially released yet, but Tombi supports `v1.1.0-preview`.

Tombi's formatter shows its true power with `v1.1.0`.
While many existing systems ([Cargo.toml](https://doc.rust-lang.org/cargo/reference/manifest.html), [pyproject.toml](https://packaging.python.org/en/latest/guides/writing-pyproject-toml/)) use `v1.0.0`. 

<Note>
To use `v1.1.0-preview`, you need to specify the configuration in `tombi.toml`.
</Note>

<Note>
`tombi.toml` supports `v1.1.0-preview` as the default version.
</Note>

## TOML Version Control

Tombi determines the TOML version for the target file in the following order of priority.

- The value of `toml-version` specified in `[[schemas]]` of Config
- The value specified in `x-tombi-toml-version` on the root object of JSON Schema
- The value of `toml-version` specified in the root of Config
