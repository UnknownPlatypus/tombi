# Linter


## CLI

Tombi's linter can be run via the CLI.

```bash
# Lint all TOML files in the current directory
tombi lint

# Lint a single file
tombi lint path/to/file.toml

# Lint multiple files
tombi lint path/to/file1.toml path/to/file2.toml

# Lint all TOML files in a directory
tombi lint .
```

## VSCode

Tombi's linter can be run via the VSCode extension.
