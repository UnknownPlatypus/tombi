# Basic Configuration

Learn how to configure Tombi using the `tombi.toml` configuration file.

## Configuration File

Tombi uses a TOML configuration file named `tombi.toml`. Place this file in your project root directory.

## Basic Structure

```toml
toml-version = "1.0.0"

[format]

[lint]

[server]
completion.enabled = true

[schema]
[schema.catalog]
path = "https://www.schemastore.org/api/json/catalog.json"

[[schemas]]
path = "https://example.com/schema.json"
include = ["my.toml"]
```
