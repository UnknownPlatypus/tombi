# Installation

Tombi can be installed through various package managers. Choose the method that best suits your needs.

## Prerequisites

- Node.js 16.0 or later (for npm installation)
- Rust 1.56 or later (for cargo installation)

## Installation Methods

### Using Pipx

```shell
pipx install tombi
```

### Manual Installation

1. Download the latest release from our [GitHub Releases](https://github.com/tombi-toml/tombi/releases) page
2. Extract the archive
3. Add the binary to your system's PATH

## Verifying Installation

After installation, verify that Tombi is properly installed:

```shell
tombi --version
```

## Editor Extensions

Tombi provides extensions for various editors:

- VS Code: Install from the [VS Code Marketplace](https://marketplace.visualstudio.com)
