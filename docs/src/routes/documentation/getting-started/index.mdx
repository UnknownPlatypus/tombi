# Quick Start Guide

Get started with Tombi in minutes! This guide will help you install and configure Tombi for your project.

## Installation

```shell
# Using pip
pip install -g tombi
```

For detailed installation instructions, see the [Installation Guide](/getting-started/installation).

## Basic Usage

1. Format a TOML file:
```shell
tombi format path/to/file.toml
```

2. Lint a TOML file:
```shell
tombi lint path/to/file.toml
```

## Configuration

Create a `tombi.toml` in your project root:

```toml
toml-version = "1.0"

[format]

[lint]

[[schemas]]
path =
```

For more configuration options, see the [Configuration Guide](/getting-started/configuration).
